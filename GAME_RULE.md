# 게임 규칙: 과일 받기 (Fruit Catcher)

## 🎮 게임 제목
**Fruit Catcher (과일 받기 대작전)**

## 📝 게임 설명
하늘에서 떨어지는 맛있는 과일들을 바구니로 받아내는 게임입니다.
몸을 움직여 바구니를 이동시키세요!
폭탄을 받으면 게임이 종료되니 조심해야 합니다.

## 🎯 게임 목표
- 제한 시간(60초) 동안 최대한 많은 과일을 받아 점수를 획득하세요.
- 폭탄은 절대 받지 마세요!

## 🕹️ 게임 규칙

### 기본 규칙
1. **화면 구성**: 화면은 왼쪽(Left), 가운데(Center), 오른쪽(Right) 3개의 라인으로 나뉩니다.
2. **조작**: 나의 포즈에 따라 바구니가 해당 라인으로 이동합니다.
3. **아이템**:
    - 🍎 **사과 (Apple)**: 획득 시 +100점
    - 🍊 **오렌지 (Orange)**: 획득 시 +200점 (더 빠르게 떨어짐)
    - 💣 **폭탄 (Bomb)**: 획득 시 **게임 오버**

### 포즈 목록 (Teachable Machine 학습 Classes)
| 포즈(Class) 이름 | 설명 | 동작 예시 |
| :--- | :--- | :--- |
| **Left** | 바구니를 왼쪽으로 이동 | 고개를 왼쪽으로 갸웃하거나 왼손 들기 |
| **Center** | 바구니를 가운데로 이동 | 정면 바라보기 (기본) |
| **Right** | 바구니를 오른쪽으로 이동 | 고개를 오른쪽으로 갸웃하거나 오른손 들기 |

**주의**: 학습 시 Class 이름을 위와 똑같이(`Left`, `Center`, `Right`) 설정해야 코드가 정상 작동합니다.

## 📊 난이도 및 레벨
- **레벨 1 (0~500점)**: 사과만 천천히 떨어짐
- **레벨 2 (500~1500점)**: 오렌지 등장, 속도 조금 빨라짐
- **레벨 3 (1500점~)**: 폭탄 등장, 속도 매우 빨라짐

## 🛠️ 개발 기능 요구사항 (AI 참고용)
1. `gameEngine.js`에서 3개 레인 시스템 구현
2. 아이템 낙하 애니메이션 (JS `requestAnimationFrame` 사용)
3. 충돌 감지 (바구니 위치와 아이템 위치 비교)
